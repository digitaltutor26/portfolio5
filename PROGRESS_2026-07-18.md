## 프로젝트 개요

중학교 국어 수행평가 자동 채점 보조 도구.  
교사가 루브릭을 설정하고 학생 제출물을 입력하면 Ollama(로컬 AI)가 1차 채점하고, 교사가 최종 검토·승인하는 구조.

## 현재 아키텍처

```
브라우저 (GitHub Pages)
    ↓ fetch
Ollama (localhost:11434) ← 각 PC에 설치 필요
    ↓ 채점 결과 JSON
브라우저 → 교사 검토/수정 → 승인
    ↓
localStorage 저장 + CSV 내보내기
```

### 평가 흐름
1. 루브릭 기준 설정 (기준명, 배점, 채점 기준 설명)
2. 학생 제출물 입력 (직접 입력 or .txt/.pdf 업로드)
3. 자동 평가 실행 → Ollama가 루브릭 기준에 따라 채점
4. 교사 검토 → AI 점수 확인 후 수정
5. 승인 → 리포트/CSV 내보내기

## 미결 사항 / 알려진 제약

- Ollama URL은 CSP 정책상 `localhost:11434` 또는 `127.0.0.1:11434`만 허용. 원격 Ollama 서버 사용 시 CSP 수정 필요
- PDF 업로드는 `file://` 프로토콜에서 불가 → 반드시 `http://localhost:3000`으로 접속
- Ollama 소형 모델(4b)의 한국어 주관적 글쓰기 평가 품질은 참고 수준. 교사 최종 판단 필수
- `reviewPolicy` select UI는 현재 미연결 상태 (항상 전체 교사 승인 방식으로 동작)
