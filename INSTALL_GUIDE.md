# Auto1 수행평가 평가 시스템 — 설치 가이드 (Windows)

이 가이드는 Auto1을 Windows PC에서 처음 설치하고 실행하는 절차를 안내합니다.

---

## 필수 설치 항목

| 항목 | 용도 | 용량 |
|------|------|------|
| Node.js | 앱 로컬 서버 실행 | 약 30MB |
| Ollama | AI 평가 엔진 (로컬 LLM) | 약 50MB |
| AI 모델 (gemma3:4b) | 실제 채점 수행 | 약 3GB |

---

## 1단계: Node.js 설치

1. 브라우저에서 `https://nodejs.org` 접속
2. **LTS 버전** 다운로드 버튼 클릭 (예: 20.x.x LTS)
3. 다운로드된 `.msi` 파일 실행 → 기본 설정 그대로 **Next** 클릭하여 설치
4. 설치 완료 후 확인:
   ```
   Windows + R → cmd 입력 → Enter
   node --version
   ```
   `v20.x.x` 같은 버전 숫자가 나오면 성공입니다.

---

## 2단계: Ollama 설치

1. 브라우저에서 `https://ollama.com` 접속
2. **Download for Windows** 버튼 클릭
3. 다운로드된 `OllamaSetup.exe` 실행 → 설치 진행
4. 설치 완료 후 시스템 트레이(우측 하단)에 Ollama 아이콘이 생기면 정상입니다.

> **참고:** Ollama는 설치 후 자동으로 백그라운드에서 실행됩니다.

---

## 3단계: AI 모델 다운로드

명령 프롬프트(cmd)를 열고 아래 명령을 실행합니다:

```
ollama pull gemma3:4b
```

- 약 **3GB** 파일을 다운로드합니다. 시간이 걸릴 수 있습니다.
- 완료 메시지가 나오면 설치 완료입니다.

### 대안 모델 (PC 사양에 따라 선택)

| 모델 | 명령 | 크기 | 필요 RAM | 속도 |
|------|------|------|---------|------|
| gemma3:4b | `ollama pull gemma3:4b` | 3GB | 8GB | 보통 |
| llama3.2:3b | `ollama pull llama3.2:3b` | 2GB | 8GB | 빠름 |
| gemma3:12b | `ollama pull gemma3:12b` | 8GB | 16GB | 느림·정확 |

> RAM이 8GB 미만이면 `llama3.2:3b` 를 권장합니다.

---

## 4단계: 앱 파일 준비

1. 제공받은 `auto1` 폴더를 적당한 위치에 저장합니다 (예: `C:\auto1`)
2. 폴더 안에 `app` 폴더와 `package.json` 파일이 있어야 합니다.

---

## 5단계: 앱 실행

### 매번 사용할 때마다 아래 순서로 실행합니다

**① Ollama 실행 확인**
- 시스템 트레이(우측 하단)에 Ollama 아이콘이 있으면 실행 중입니다.
- 없으면 시작 메뉴에서 **Ollama** 검색 후 실행합니다.

**② 명령 프롬프트에서 앱 서버 시작**
```
cd C:\auto1
npx serve app
```
처음 실행 시 `serve` 패키지를 자동 설치합니다 (인터넷 필요, 1회만).

**③ 브라우저에서 접속**
```
http://localhost:3000
```
위 주소를 브라우저 주소창에 입력합니다.

> ⚠️ 파일 탐색기에서 `index.html`을 직접 더블클릭하면 PDF 업로드와 AI 평가가 동작하지 않습니다. 반드시 `http://localhost:3000`으로 접속하세요.

---

## 6단계: 첫 사용 설정

1. 브라우저에서 앱을 열면 **평가 설정** 패널이 보입니다.
2. 하단의 **AI 평가 설정 (Ollama)** 섹션 확인:
   - **Ollama 서버 주소**: `http://localhost:11434` (기본값 그대로)
   - **모델명**: 다운로드한 모델명 입력 (예: `gemma3:4b`)
3. 루브릭의 **채점 기준 설명** 항목을 과제에 맞게 수정합니다.
4. 학생 제출물을 입력하고 **자동 평가 실행** 버튼을 누릅니다.
5. AI 평가 결과를 검토하고 **교사 최종 점수**를 수정한 뒤 **승인** 버튼을 누릅니다.

## 문제 해결

### "Ollama 서버에 연결할 수 없습니다" 오류
- 시스템 트레이에 Ollama 아이콘이 있는지 확인합니다.
- 없으면 시작 메뉴에서 Ollama를 다시 실행합니다.
- cmd에서 `ollama list` 명령을 실행해 모델이 설치되어 있는지 확인합니다.

### "응답 시간 초과" 오류
- PC RAM이 부족할 수 있습니다. 더 작은 모델(`llama3.2:3b`)을 사용하거나, 다른 프로그램을 닫고 재시도합니다.

### PDF 파일이 업로드되지 않음
- `file://`로 시작하는 주소로 접속 중인 경우입니다. `http://localhost:3000`으로 접속했는지 확인합니다.

### 앱 서버가 시작되지 않음
- Node.js가 설치되어 있는지 확인합니다: cmd에서 `node --version`
- `cd C:\auto1` 명령으로 폴더 위치를 정확히 이동했는지 확인합니다.

### 포트 3000이 이미 사용 중
```
npx serve app -p 3001
```
위 명령으로 다른 포트를 사용하고, 브라우저에서 `http://localhost:3001`로 접속합니다.
